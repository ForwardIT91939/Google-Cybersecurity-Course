Activity: Practice writing Python code
---------------------------------------
In this lab, you will open a notebook environment to practice writing Python code. 
You’ll be presented with a security scenario to explore throughout the lab. 
You’ll become familiar with working in a notebook environment, writing code comments in Python, and displaying strings with the print() function.
---------------------------------------
Tasks:
Learn about the capabilities of a notebook environment
Write code comments in Python
Display strings with the print() function
